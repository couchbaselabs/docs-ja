CLIリファレンス

Couchbase Serverコマンドラインインタフェース（CLI） ツールはクラスタ、サーバ、vBuckets、XDCRなどの管理や監視を提供します。

Related information

  1. CLI overview
    The command-line tools perform different functions and operations, these CLI tools are described individually.
  2. couchbase-cli tool
    The couchbase-cli tool is used to perform operations on an entire cluster.
  3. cbanalyze-coreツール
    cbanalyze-coreツールはコアダンプデータの解析や分析に使用します。
  4. cbbackupツール
    cbbackupツールは、稼働中の全クラスタ・全バケット・単一ノード・正常稼働中ノード上の単一バケット のコピーを作成します。
  5. cbcollect_info tool
    The cbcollect_info tool provides detailed statistics for a specific node.
  6. cbdocloader tool
    The cbdocloader tool is used to load a group of JSON documents in a given directory or in a single .zip file.
  7. cbepctlツール
    cbepctlツールはvBucketのステータス、設定、メモリとディスク永続化の挙動をコントロールするために利用します。
  8. cbhealthchecker tool
    The cbhealthchecker tool generates a health report named Cluster Health Check Report for a cluster.
  9. cbreset_password tool
    The cbreset_password tool is used to reset an administrative or user read-only password.
  10. cbrestore tool
    The cbrestore tool restores data from a file to an entire cluster or to a single bucket in the cluster.
  11. cbstats tool
    The cbstats tool is used to get node and cluster-level statistics about performance and items in storage.
  12. cbtransfer tool
    The cbtransfer tool is used to transfer data between clusters and to/from files.
  13. cbworkloadgen tool
    The cbworkloadgen tool is used to generate random data and perform read/writes.
