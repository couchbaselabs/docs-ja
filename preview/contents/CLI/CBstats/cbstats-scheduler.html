    Scheduler

    The cbstats scheduler command provides scheduler task statistics.

    CLI command and parameters

    cbstats scheduler

    Syntax

    Request syntax:

    cbstats [localhost]:11210 scheduler

    Example

    Example request:

    cbstats 10.5.2.117:11210 scheduler
