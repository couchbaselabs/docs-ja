    Allocator

    The cbstats allocator command provides information on memory management.

    CLI command and parameters

    cbstats allocator

    Syntax

    Request syntax:

    cbstats [localhost]:11210 allocator

    Example

    Example request:

    cbstats 10.5.2.54:11210 allocator
