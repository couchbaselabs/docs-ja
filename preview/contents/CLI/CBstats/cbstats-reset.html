    Reset

    The cbstats reset command resets the following reset and reset histogram statistics.

    CLI command and parameters

    cbstats reset

    Syntax

    Request syntax:

    cbstats [localhost]:11210 reset

    Example

    Example request:

    cbstats 10.5.2.54:11210 reset
