    Failovers

    The cbstats failovers command provides ID, sequence, and number of entries information.

    CLI command and parameters

    cbstats failovers

    Syntax

    Request syntax:

    cbstats [localhost]:11210 failovers

    Example

    Example request:

    cbstats 10.5.2.54:11210 failovers

    Response

    Example response:
