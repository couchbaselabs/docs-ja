    Runtimes

    The cbstats runtimes command provides various information about runtime statistics.

    CLI command and parameters

    cbstats runtimes

    Syntax

    Request syntax:

    cbstats [localhost]:11210 runtimes

    Example

    Example request:

    cbstats 10.5.2.117:11210 runtimes
